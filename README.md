# Corporate Lease Hub

Corporate Lease Hub is an Executive Relocation & Corporate Housing Solution designed to streamline high-level corporate leasing, temporary housing placement, and internal property/vendor management.

Built for enterprise clients, relocation coordinators, and premium housing providers.

---

## 🚀 Vision

To be trusted as the premier centralized platform for executive-level corporate relocation and housing operations — combining placement, lease management, vendor coordination, and internal workflow automation into one secure system.

---

## 🏢 Core Solutions

- Executive relocation coordination
- Corporate housing placement
- High-tier lease management
- Internal property management tools
- Vendor onboarding and oversight
- Secure authentication & account management
- Payment processing integration (Stripe)

---

## 🧱 Tech Stack

Frontend:
- Vite
- React
- TypeScript

Backend:
- Node.js
- Express

Database:
- PostgreSQL
- Drizzle ORM

Infrastructure:
- Hosted on Render
- Database hosted on Neon
- Payments via Stripe (in progress)

---

## 🔐 Security & Authentication

- Secure session management
- Password reset functionality
- Environment variable protection
- Production-ready deployment architecture

---

## 🛠 Development

### Run backend (development)
